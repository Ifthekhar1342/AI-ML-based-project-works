# Generative AI for 3D CAD Model Reconstruction and Latent-Space Morphing

## Objective
To reconstruct 3D CAD models from input data and interpolate between different 3D CAD designs in a meaningful way.

## Dataset
<p align="justify">
The dataset of this work is taken from the <a href="https://drive.google.com/drive/u/0/folders/1Oenec-8J2A5QJKMSCWkW_a3keiLYfkkL">ModelNet10</a>, which contains 10 categories of CAD models (i.e. chair, bed) along with .off files.
</p>
